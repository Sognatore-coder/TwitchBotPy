# GamingBotPy 🎮

**GamingBotPy** — это производительный Telegram-бот, интегрированный с Twitch API. Бот позволяет отслеживать состояние игровых стримов, собирать аналитику каналов и автоматизировать взаимодействие между платформами Telegram и Twitch.

---

## 🚀 Ключевые функции

* **Интеграция с Twitch API (`api.py`)**: Получение актуальных данных о стримах, статусе онлайна, количестве зрителей и информации о каналах в реальном времени.
* **Мониторинг и парсинг (`scraper.py`)**: Сбор дополнительных открытых данных для аналитики.
* **Мультиязычность (`translator.py`)**: Автоматический перевод уведомлений или описания стримов для русскоязычной аудитории.
* **Логирование (`logger.py`)**: Запись работы системы и обработка ошибок в изолированные файлы для удобной отладки.

---

## 🔐 Получение секретов Twitch API

Для работы бота требуются ключи доступа разработчика Twitch (`Client ID` и `Client Secret`). 

1. Перейдите в **[Консоль разработчика Twitch (Twitch Developer Console)](https://twitch.tv)**.
2. Войдите под своей учетной записью Twitch (желательно включить двухфакторную аутентификацию).
3. Нажмите кнопку **Register Your Application** (Зарегистрировать приложение).
4. Укажите параметры:
   * **Name**: Название вашего бота (например, `MyGamingBot`).
   * **OAuth Redirect URLs**: Укажите `http://localhost`, если используете бота локально.
   * **Category**: Выберите `Application Bot` или `Chat Bot`.
5. Нажмите **Create**. В списке приложений откройте созданное и скопируйте **Client ID**.
6. Нажмите **New Secret**, чтобы сгенерировать **Client Secret** (сохраните его, он показывается один раз).

## 🛠 Процесс запуска

### 1. Клонирование и подготовка
Убедитесь, что у вас установлен Python версии 3.10 или выше. Клонируйте проект и перейдите в его директорию:
```bash
git clone https://github.com/Sognatore-coder/TwitchBotPy.git
cd GamingBotPy
```

### 2. Настройка окружения и зависимостей
Установите все необходимые библиотеки
```bash
pip install "название библиотеки"
```

### 3. Конфигурация токенов
Скопируйте пример конфигурационного файла и заполните его своими данными:
```bash
cp config.example.py config.py
```
Откройте созданный `config.py` и вставьте туда:
* Токен вашего Telegram-бота (полученный у `@BotFather`)
* Ключи `Client ID` и `Client Secret` из кабинета Twitch.


### 4. Запуск бота
```bash
python bot.py
```
